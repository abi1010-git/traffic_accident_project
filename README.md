# Traffic Accident Analysis Project

This project analyzes traffic accident data using Python, pandas, and matplotlib.

## Folder Structure

- `data/` - Put the raw accident dataset here as a CSV or Excel file.
- `notebooks/analysis.ipynb` - Main exploratory analysis notebook.
- `src/traffic_analysis.py` - Helper functions used by the notebook.

## How To Run

1. Put your accident dataset in the `data/` folder.
2. Open `notebooks/analysis.ipynb` in VS Code.
3. Select the project virtual environment as the notebook kernel.
4. Run the notebook cells from top to bottom.

The notebook will load the first `.csv`, `.xlsx`, or `.xls` file it finds in `data/`.
